# 💻 **프로젝트 소개**
🕊 EarlyBirdy 어플리케이션은 미라클 모닝을 도와주는 어플리케이션으로 알람, 오늘의 목표 설정 기능을 제공합니다.
기상, 오늘의 목표 달성으로 경험치를 얻어 순위 경쟁에 도전할 수 있습니다!

- **어플리케이션 이름** : EarlyBirdy
- **어플리케이션 소개** :  'EarlyBirdy (얼리버디)'는 '일찍 일어나는 사람' 또는 '조기 기상자'를 뜻하는 영어 신조어입니다.
